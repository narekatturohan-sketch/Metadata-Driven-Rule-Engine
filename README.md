# Metadata-Driven Rule Engine

## 📌 Overview
This project implements a metadata-driven rule engine for banking validations.
Rules are stored in Oracle 23c and applied dynamically through a FastAPI service.
It supports regex, length, and range validations, making the system flexible and easy to extend.

---

## 🛠️ Tech Stack
- **Language:** Python 3.14
- **Framework:** FastAPI
- **Database:** Oracle 23c Free Edition
- **Libraries:** oracledb, regex, uvicorn
- **Tools:** Git, Postman
